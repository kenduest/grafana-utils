.TH GRAFANA-UTIL-CHANGE 1 "2026-04-03" "grafana-util 0.7.3" "User Commands"
.SH NAME
grafana\-util\-change \- review\-first sync, preflight, audit, and apply workflows
.SH SYNOPSIS
.B grafana-util change [\fISUBCOMMAND\fR] [\fIOPTIONS\fR]
.SH DESCRIPTION
review\-first sync workflows with optional live Grafana fetch and apply paths.
.PP
when you need to summarize desired resources, plan against live state, review a plan, apply a reviewed plan, audit drift, or build bundle and promotion preflight documents.
.SH SUBCOMMANDS
.TP
.B summary
summarize local desired sync resources.
.TP
.B plan
build a staged sync plan from desired and live state.
.TP
.B review
mark a staged sync plan as reviewed.
.TP
.B apply
build a gated apply intent from a reviewed sync plan, and optionally execute it live.
.TP
.B audit
audit managed Grafana resources against a checksum lock and current live state.
.TP
.B preflight
build a staged sync preflight document from desired resources and optional availability hints.
.TP
.B assess\-alerts
assess alert sync specs for candidate, plan\-only, and blocked states.
.TP
.B bundle
package exported dashboards, alerting resources, datasource inventory, and metadata into one local source bundle.
.TP
.B bundle\-preflight
build a staged bundle\-level sync preflight document from a source bundle and target inventory.
.TP
.B promotion\-preflight
build a staged promotion review handoff from a source bundle and target inventory.
.SH EXAMPLES
.EX
grafana-util change summary --desired-file ./desired.json
.EE
.EX
grafana-util change plan --desired-file ./desired.json --fetch-live --profile prod
.EE
.EX
grafana-util change apply --plan-file ./sync-plan-reviewed.json --approve --execute-live --profile prod
.EE
.EX
grafana-util change summary --desired-file ./desired.json --output json
.EE
.EX
grafana-util change plan --desired-file ./desired.json --live-file ./live.json
.EE
.EX
grafana-util change plan --desired-file ./desired.json --fetch-live --profile prod --allow-prune --output json
.EE
