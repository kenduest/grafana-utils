.SH NAME
grafana\-util\-profile \- manage repo\-local grafana\-util profile configuration
.SH SYNOPSIS
.B grafana-util profile [\fISUBCOMMAND\fR] [\fIOPTIONS\fR]
.SH DESCRIPTION
list, inspect, add, and initialize repo\-local grafana\-util profiles.
.PP
when you want to keep Grafana connection defaults in the current checkout and reuse them with \-\-profile.
.SH SUBCOMMANDS
.TP
.B list
list profile names from the resolved grafana\-util config file.
.TP
.B show
show the selected profile as text, table, csv, json, or yaml.
.TP
.B add
create or replace one named profile without hand\-editing grafana\-util.yaml.
.TP
.B example
print a comment\-rich reference config that operators can copy and adapt.
.TP
.B init
initialize grafana\-util.yaml in the current working directory.
.SH EXAMPLES
.EX
grafana-util profile list
.EE
.EX
grafana-util profile show --profile prod --output-format yaml
.EE
.EX
grafana-util profile show --profile prod --output-format json
.EE
.EX
grafana-util profile show --profile prod --show-secrets --output-format yaml
.EE
.EX

.EE
